MedChat LLM
Multimodal Medical Chatbot · Oct 2024

MedChat is a multimodal medical chatbot combining Llama 2 with Retrieval-Augmented Generation to deliver accurate, context-grounded clinical answers — significantly reducing hallucinations.

Core Features

  • Llama 2 as backbone LLM for medical question answering
  • LangChain-powered RAG pipeline grounding answers in real documents
  • ChatGPT Vision API for analyzing medical images (X-rays, scans)
  • FastAPI backend for clean, scalable API serving

Outcome

Reduced hallucinations via RAG; end-to-end prototype completed.

Tech Stack

Llama 2 · LangChain · FastAPI · ChatGPT Vision API · Python · Vector DB
